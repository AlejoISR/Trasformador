# 🛠️ SuperConversor CLI

¡Bienvenido a **SuperConversor**! Una herramienta de línea de comandos rápida, ligera y fácil de usar diseñada para automatizar la conversión y el renombrado de archivos directamente desde tu terminal.

Olvídate de páginas web llenas de anuncios para convertir un PDF o de renombrar archivos uno por uno.

## ✨ Características Principales

* **🐍 Renombrado Masivo (Py a Txt):** Cambia la extensión de todos tus scripts `.py` a `.txt` en un instante. Ideal para compartir código por correo sin que los filtros de seguridad lo bloqueen.
* **📄 Conversión de PDF a Word:** Transforma rápidamente cualquier archivo `.pdf` en un documento editable `.docx`.
* **⚡ 100% Portable:** Empaquetado en un único archivo `.exe`. Puedes llevarlo en un pendrive y usarlo en cualquier ordenador sin necesidad de instalar Python.

## 🚀 Cómo usarlo

Si has descargado el archivo `.exe`:

1. Coloca el archivo `conversor.exe` en la misma carpeta donde tengas los archivos que quieres modificar.
2. Haz doble clic sobre él (o ejecútalo desde tu CMD).
3. Sigue las instrucciones del menú interactivo. ¡Así de simple!

Si prefieres usar el código fuente en Python:

```bash
# 1. Clona este repositorio
git clone https://github.com/AlejoISR/Trasformador.git

# 2. Instala las dependencias
pip install -r requirements.txt

# 3. Ejecuta el script
python conversor.py
